Je třeba si vytvořit envoromentální soubor cypress.env.json s těmito hodnotami: 

{
    "TEST_EMAIL": "",
    "TEST_PASSWORD": "",
    "TEST_USERNAME": ""
}

Do hodnot si napište údaje s kterými chcete testovat, je to jedno, ale je to good practice.




Projekt obsahuje testování hlavní stránky, registrace a repozitářů ( pro příklad jsem testoval můj ).
